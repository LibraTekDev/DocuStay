Pre-requisite: Reset the database
Since the schema has new columns and the seed data is completely new (50 states instead of 5), you need to recreate the tables. The easiest way with PostgreSQL:

# Drop and recreate the jurisdiction-related tables so create_all picks up new columns
# and the seed function re-populates with all 50 states
psql -U postgres -d docustay_demo -c "DROP TABLE IF EXISTS jurisdiction_zip_mappings, jurisdiction_statutes, jurisdictions, region_rules CASCADE;"
Then restart the backend server. On startup, create_all will recreate the tables with the new columns, and the seed functions will populate all 50 states.
